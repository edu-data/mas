# 창의성 (Creativity)

## 평가 영역

**배점: 5점**

창의성은 교사가 수업에서 창의적이고 독창적인 교수 방법, 자료, 활동을 활용하는 능력을 평가합니다.

## 세부 평가 기준

### 1. 수업 창의성 (5점)

- **우수 (5점)**: 독창적인 교수 전략, 참신한 자료 활용, 예상치 못한 전개로 학생들의 흥미 극대화
- **양호 (4점)**: 일부 창의적 요소가 있으나 전반적으로 일반적인 수업
- **보통 (3점)**: 기본적인 수업 구성, 창의적 요소 부족
- **미흡 (1-2점)**: 매우 전통적이고 단조로운 수업

**창의적 수업 요소 예시:**

- 게이미피케이션 활용 (퀴즈, 경쟁, 보상 시스템)
- 역할극, 시뮬레이션 활동
- 디지털 도구 활용 (VR, AR, 인터랙티브 앱)
- 스토리텔링 기법
- 프로젝트 기반 학습
- 거꾸로 교실 (Flipped Classroom)

## 교육학적 근거

### Robinson의 창의성 교육

창의성은 "가치 있는 독창적 아이디어를 생성하는 과정"이며, 교사는 창의적 사고를 모델링해야 합니다.

### STEAM 교육

과학(S), 기술(T), 공학(E), 예술(A), 수학(M)을 융합한 창의적 문제해결 중심 교육.

### 디자인 씽킹 (Design Thinking)

1. 공감 (Empathize)
2. 정의 (Define)
3. 아이디어 도출 (Ideate)
4. 프로토타입 (Prototype)
5. 테스트 (Test)

### 창의적 수업의 특징

- 열린 문제 (Open-ended Problems)
- 학생 선택권 부여
- 실패를 허용하는 분위기
- 다양한 해결책 탐색

## 피드백 가이드

### 강점 피드백

- "학생들의 투표 결과를 활용한 모둠 구성은 창의적인 접근입니다."
- "게이미피케이션 요소를 도입하여 학생들의 참여도를 높였습니다."

### 개선 피드백

- "일상적인 사례나 최신 이슈를 수업에 연결하면 더 흥미롭습니다."
- "학생들이 스스로 문제를 정의하고 해결책을 찾는 활동을 추가해 보세요."
