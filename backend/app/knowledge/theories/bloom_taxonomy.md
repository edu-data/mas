# Bloom의 교육목표 분류학 (Bloom's Taxonomy)

## 개요

1956년 Benjamin Bloom이 제안한 인지적 학습 목표의 위계적 분류 체계.

## 인지적 영역 (개정판, 2001)

| 수준 | 설명 | 수업 활동 예시 |
|------|------|----------------|
| 기억하기 | 정보 회상, 인식 | 용어 정의, 사실 나열 |
| 이해하기 | 의미 파악, 설명 | 요약, 비교, 분류 |
| 적용하기 | 새로운 상황에 적용 | 문제 해결, 실험 |
| 분석하기 | 구성요소로 분해 | 원인 분석, 패턴 발견 |
| 평가하기 | 기준에 따른 판단 | 비판적 검토, 토론 |
| 창조하기 | 새로운 것 생성 | 설계, 발명, 창작 |

## 수업 적용

### 학습목표 설정

- 고차원적 사고력(HOT) 목표 포함 권장
- "~를 기억한다" 외에 "~를 분석한다", "~를 창조한다" 활용

### 발문 전략

- **저차원**: "무엇", "언제", "누가" (사실 확인)
- **고차원**: "왜", "어떻게", "만약" (분석, 평가, 창조)

## 평가 연계

### 수업 전문성 (20점)

- 학습목표가 다양한 인지 수준을 포함하는지 평가
- 고차원 사고력 활동 포함 여부 확인

### 학생 참여 (15점)

- 발문이 다양한 인지 수준을 자극하는지 평가
- 열린 질문 활용 빈도 분석

## 참고문헌

- Bloom, B. S. (1956). Taxonomy of Educational Objectives
