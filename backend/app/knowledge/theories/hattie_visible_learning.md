# Hattie의 가시적 학습 (Visible Learning)

## 개요

John Hattie의 800개 이상 메타분석 연구에 기반한 효과적인 교수법 연구.

## 핵심 개념

### 효과 크기 (Effect Size)

교육적 개입의 효과를 측정하는 표준화된 지표.

| 효과 크기 | 해석 |
|-----------|------|
| d > 0.40 | 유의미한 효과 |
| d > 0.60 | 강한 효과 |
| d > 1.00 | 매우 강한 효과 |

## 고효과 교수 전략 (d > 0.70)

### 1. 피드백 (d = 0.73)

- 즉각적, 구체적, 개선지향적
- "잘했어요" 대신 "이 부분이 좋았는데, 여기는 이렇게 하면 더 좋겠어요"

### 2. 교사-학생 관계 (d = 0.72)

- 신뢰와 존중의 분위기
- 학생 이름 불러주기, 관심 표현

### 3. 형성 평가 (d = 0.90)

- 학습 과정 중 지속적 확인
- "지금까지 이해가 됐나요?"

### 4. 교사 명료성 (d = 0.75)

- 명확한 학습 목표
- 성공 기준 제시
- "오늘 수업이 끝나면 ___를 할 수 있어요"

### 5. 협동 학습 vs 개별 학습 (d = 0.59)

- 구조화된 협동 학습
- 역할 분담, 개인 책무성

### 6. 자기평가/자기조절 (d = 0.62)

- 학생 스스로 학습 점검
- "내가 알고 있는 것/모르는 것은?"

## 평가 연계

### 학생 참여 (15점)

- 피드백 제공 빈도와 질
- 형성 평가 활용

### 수업 태도 (15점)

- 학생과의 긍정적 상호작용
- 명확한 의사소통

### 교수학습 방법 (20점)

- 효과적인 교수 전략 활용
- 협동 학습 설계

## 피드백 가이드

### 효과적인 피드백의 3가지 질문

1. 어디로 가고 있는가? (목표)
2. 어떻게 가고 있는가? (진행)
3. 다음에 어디로 갈 것인가? (발전)

### 피드백 수준

- **과제 수준**: 정오 확인
- **과정 수준**: 전략 안내
- **자기조절 수준**: 메타인지 촉진
- **자아 수준**: 칭찬 (효과 낮음)
